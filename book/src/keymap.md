# Keymap


## Movement

| Key | Description |
|-----|-----------|
| h   | move left |
| j   | move down |
| k   | move up |
| l   | move right |
| w   | move next word start |
| b   | move previous word start |
| e   | move next word end |

| Key | Description |
|-----|-----------|
| a   | Append to end of selection |
